# Bash

## Super Basic Common Commands

```
// What shell am I using

$ echo $SHELL

// List whats in a directory

$ ls

// Move around the file system

$ cd

// Display the current directory

$ pwd 

// List out options for a program

$ man <program>

// remove empty folder
$ rmdir folder-name

// remove folder w/ contents
$ rm –r folder-name

```

| Function     | Command |
| ------------ | ------- |
| delete file  | rm      |
