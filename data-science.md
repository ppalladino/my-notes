# Data Science

### Vocabulary

| Term                   |                                                                                                                                   |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| clustering             | grouping data points into categories                                                                                              |
| sample size calculator | online calculator to help determine how large a sample size needs to be for a giving data set, error, and confidence requirements |
| data wrangling         | cleaning and organizing data                                                                                                      |

### Tools&#x20;

|                                          |                                                                                                                                                                      |
| ---------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Importing, cleaning, organizing datasets |  [https://pandas.pydata.org/pandas-docs/stable/](https://pandas.pydata.org/pandas-docs/stable/) - Python lib **Pandas** for importing, cleaning, organizing datasets |
| Statistical analysis                     | Python module known as **NumPy** to calculate descriptive statistics values.                                                                                         |
| Visualizing data                         | Python: **Matplotlib** and **Seaborn**, JS: **D3**                                                                                                                   |
