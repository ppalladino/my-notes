# VS Code

## Symbols Key

| **Symbol** | **Meaning**           |
| ---------- | --------------------- |
| ⇧          | shift                 |
| ^          | control               |
| ⌥          | option                |
|  ⌘         | command               |
| ←↑→↓       | left, up, right, down |
| ↕          | up/down               |
|            |                       |

## Key Bindings

| **Action**  | **Shortcut** |
| ----------- | ------------ |
| Delete line | ⇧ ⌘ K        |
| Move line   | ⌥  ↕         |
| Go Back     | ^-           |
| Go Forward  | ^⇧-          |
