### Vocabulary

| Term                   |                                                                                                                                   |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| clustering             | grouping data points into categories                                                                                              |
| sample size calculator | online calculator to help determine how large a sample size needs to be for a giving data set, error, and confidence requirements |
| data wrangling         | cleaning and organizing data                                                                                                      |

## Anaconda

Anaconda is a distribution of the Python and R programming languages for scientific computing. It takes some of the challanges out of all the package managment and config required. Uses IPython on the backend to execute code

```
// Open Anaconda Navigator
$ anaconda-navigator

// Start an Anaconda Prompt
$ anaconda prompt
```

### Jupyter Notebook

IDE environment within Anaconda for python: [https://jupyter-notebook.readthedocs.io/en/stable/](https://jupyter-notebook.readthedocs.io/en/stable/)

```
// Open a notebook
$ jupyter notebook
// Or open anaconda navigator and pen via GUI
```
